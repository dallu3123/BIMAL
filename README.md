# BIMAL
Bimanual Manipulation Learning

이 레포지토리는 **Bimanual Manipulation(양손 조작)** 연구를 위한 코드입니다.
Piper 기반 하드웨어와 ALOHA(teleoperation & imitation learning framework)를 참고하여,
양손 로봇 조작을 위한 **Imitation Learning(IL)** 및 **Reinforcement Learning(RL)** 기법을 탐구합니다.

## 환경 설정
```bash
conda env create -f environment.yml
conda activate bimanual
cd <path to this repository>
```
